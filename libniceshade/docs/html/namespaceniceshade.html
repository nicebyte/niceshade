◆ stage_mask_bit

enum niceshade::stage_mask_bit

Enumerates bits corresponding ti programmable shader stages.

The values of individual enumerants can be combined together using the logical OR operation to form a bitmask indicating which stages of the pipeline a particular resource is visible/used from.
