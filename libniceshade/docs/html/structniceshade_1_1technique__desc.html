std::vector<std::pair<std::string, std::string> > niceshade::technique_desc::additional_metadata

User-provided metadata.

This is only really useful when the techniques are specified directly inside the HLSL code using the special //T: comment. In that case, the additional metadata shall be parsed out and made available to the application, which can then use it for its own purposes (for example, enable or disable depth write for particular techniques based on user metadata values). See the README for more details on inline technique definition.
